Apa Itu HTML?

HTML adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau Markup ini, nanti akan kita sebut sebagai Tag. Penanda atau markup ini, nanti Kita akan sebut dengan HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. kalau diibaratkan nih.. HTML Itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar. HTML memiliki kelebihan seperti memiliki sumber komunitas yang cukup besar dengan pengguna yang luas Tidak hanya itu, kelebihan lainnya adalah HTML mudah untuk dipelajari oleh Seorang Web Developer Pemula kelebihan lainnya adalah HTML merupakan Bahasa Pemrograman yang gratis dan open source
